# 常见问题

> 来源：[有赞云常见问题](https://doc.youzanyun.com/resource/doc/3381)

## 入驻与账号

### Q: 如何进行有赞云认证？
A: 登录控制台 → 进入团队 → 点击"去认证" → 选择个人或企业认证 → 提交资料 → 等待审核（1 个工作日）

### Q: 个人认证和企业认证有什么区别？
A: 企业认证可展示更丰富的企业信息，适合第三方软件服务商。个人认证适合个人开发者自用。

### Q: 微商城已认证主体，创建应用为何还需认证？
A: 有赞云的认证体系独立于 SaaS 商家体系，需单独完成开发者认证。

### Q: 认证审核需要多久？
A: 1 个工作日内完成。若审核不通过，需重新提交资料（图片需小于 2MB）。

## 应用创建

### Q: 自用型和工具型应用有什么区别？

| | 自用型 | 工具型 |
|---|---|---|
| 目标用户 | 自己公司 | 第三方商家 |
| 上架市场 | 不需要 | 必须上架应用市场 |
| 授权方式 | 店铺后台确认授权 | 商家自行订购授权 |
| 适用场景 | 内部系统对接 | SaaS 服务商 |

### Q: 有容器和无容器应用有什么区别？

| | 无容器型 | 有容器型 |
|---|---|---|
| 代码部署 | 部署在自己服务器 | 部署在有赞云服务器 |
| 后端扩展点 | 不支持 | 支持 |
| 运维复杂度 | 高（自己维护） | 低（有赞托管） |

## 授权与 Token

### Q: 如何获取 access_token？
A: 通过 OAuth2.0 授权流程：
1. 引导商家访问授权链接
2. 商家授权后回调获取 `code`
3. 用 `code` 换取 `access_token` 和 `refresh_token`

### Q: access_token 有效期多久？如何刷新？
A: 有效期 2 小时。使用 `refresh_token`（有效期 30 天）换取新的 access_token。

## 接口调用

### Q: 调用返回 10004 无权限？
A: 该接口需要额外申请权限。进入控制台 → 应用管理 → 接口权限，申请开通后重试。

### Q: 调用返回 20001 频率超限？
A: 当前调用频率超过限制。降低调用频率，或升级 API 套餐获取更高配额。

### Q: 接口返回数据为空？
A: 检查：① token 是否有效 ② 店铺是否有数据 ③ 接口参数是否正确

### Q: 如何知道某个接口是否需要计费？
A: 进入控制台 → 应用管理 → 能力包，查看接口所属的权限包。计费接口标有红色"计费"标签。

## 数据加密

### Q: 订单中的手机号是密文，如何解密？
A: 使用加解密 API `youzan.cloud.secret.decrypt.single` 解密。详见[加解密指南](encryption.md)。

### Q: 不同店铺的手机号密文相同吗？
A: 不同。密钥是店铺维度的，存储密文时必须同时记录店铺 ID。

## 套餐与计费

### Q: API 调用如何计费？
A: 通过 API 套餐包计费。购买配套 SaaS 版本（如有赞微商城旗舰版）即赠送对应套餐。详细见 [API 计费规则](pricing.md)。

### Q: 免费额度有多少？
A: 2025-07-01 前，自用型应用赠送 50 万次/月 API + 100 万次/月消息。2025-07-01 后改为先付费后使用。

### Q: 欠费后会自动恢复吗？
A: 欠费后 API 会被限流/停用。缴清欠费或订购套餐后自动恢复。

## 有容器应用

### Q: 如何在后端扩展点中获取请求参数？
A: 后端扩展点通过 Spring Bean 接收请求，具体参数由扩展点接口定义。可参考控制台提供的示例代码。

### Q: 如何调试有容器应用？
A: 使用控制台的云调试工具，可模拟触发扩展点进行本地调试。
