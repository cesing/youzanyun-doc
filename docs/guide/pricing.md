# API 套餐与计费规则

> 来源：[有赞云API服务费规则](https://doc.youzanyun.com/resource/doc/8770)（最新调整日期：2026-02-28）

## 当前有效套餐（2026-03-10 起生效）

### 开放平台 API 套餐

| 套餐 | 配套 SaaS 版本 | 价格 | API 额度（/月） | 消息额度（/月） | 技术支持 |
|------|----------------|------|----------------|----------------|----------|
| **A 套餐** | 有赞美业专业版 | 随版本赠送（单独购买 2,580 元/年） | 50 万次 | 50 万次 | 工单咨询 |
| **B 套餐** | 有赞微商城旗舰版 / 门店旗舰版 / 连锁专业版（总部）/ 美业旗舰版 | 随版本赠送（单独购买 5,980 元/年） | 200 万次 | 200 万次 | 工单咨询 |
| **C 套餐** | 有赞微商城至尊版 / 门店至尊版 / 连锁旗舰版（总部） | 随版本赠送（单独购买 23,800 元/年） | 1,200 万次 | 1,200 万次 | 含技术支持服务-专业版 1 个月 |

> 所有套餐**不含**：大客接口、iPaaS 套餐、鹊桥接口、CRM 接口及未订购的产品功能接口。

## 附加服务

### 有赞技术支持服务 - 专业版

- **价格**：15,000 元/月（限时优惠）
- **适用**：无容器应用，配套已购 API 套餐使用
- **内容**：1v1 企业微信群，5×8 小时，2 小时响应
- **不含**：可行性分析、代码调试、该应用未拥有的 API 问题、iPaaS 对接

### 特殊插件赠送额度

| 插件类型 | 每月赠送 API | 每月赠送消息推送 |
|----------|-------------|-----------------|
| App 开店 | 1,000 万次 | 1,500 万次 |
| 半屏小程序 | 1,000 万次 | 1,500 万次 |
| 有赞云个性化定制服务 | 5,000 万次 | 7,500 万次 |
| 有赞云 iPaaS 数据互通服务 | 5,000 万次 | 7,500 万次 |

## 免费额度政策（2025-07-01 ～ 2026-06-30）

**自用型应用：**

- 2025 年 7 月 1 日前：继续赠送 50 万次/月 API + 100 万次/月消息推送
- 2025 年 7 月 1 日起：**不再赠送**，改为先付费后使用模式

**收费模式变化：**
- 原：先用后付
- 现：**先付后用** — 需先订购 API 套餐包或购买对应 SaaS 产品获得赠送套餐

---

## 欠费说明

- 应用欠费后，API 相关功能将被**限流甚至停用**（包括消息推送）
- 缴清欠费或订购/升级套餐后，功能自动恢复
- ⚠️ 套餐额度**不能抵扣历史欠费账单**

---

## 连锁店铺特别规则

- **有赞连锁至尊版（总部）**：赠送两套 23,800 元/年 C 套餐，含 2,400 万次 API + 消息推送/月
- **分店（同城云店/门店收银/区域网店）**：每分店额外赠送 5 万次 API + 5 万次消息/月，额度可叠加

## 如何选择套餐

```
商家类型                     推荐套餐
─────────────────────────────────────────────
有赞美业（专业版）           → A 套餐
有赞微商城旗舰版/门店旗舰版  → B 套餐
有赞微商城至尊版/连锁旗舰版  → C 套餐
有赞连锁至尊版               → C 套餐 + 额外一套 C 套餐
第三方软件服务商（工具型）   → 按调用量选购对应 SaaS 版本
```
