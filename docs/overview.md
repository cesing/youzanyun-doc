# 有赞云开放平台概览

## 平台定位

有赞云是有赞面向第三方开发者的**开放平台**，提供 API、消息推送和后端扩展点三类能力，帮助开发者将有赞店铺与自有系统对接，实现数据互通和业务自动化。

## 核心能力

### 1. API 能力
- **覆盖场景**：商品、订单、会员、营销、库存、物流、支付、分销等
- **接口数量**：2000+ 个（含已上线/变更中）
- **协议类型**：HTTP / Dubbo RPC
- **认证方式**：凭证式 / 签名式 / 无需认证

### 2. 消息推送
- 商家操作事件实时推送至开发者服务器
- 支持订单、售后、退款、会员等事件类型
- 无需频繁轮询，降低调用成本

### 3. 后端扩展点（有容器型应用）
- 在有赞云服务器运行代码
- 可拦截和修改有赞内部业务流程
- 适用于需要深度定制的场景

## 应用类型

```
有赞云应用
├── 自用型    → 商家自研，对接自有系统
└── 工具型    → 软件服务商，上架应用市场，供商家订购
    ├── 无容器  → 代码部署在自己服务器
    └── 有容器  → 代码部署在有赞云服务器
```

## 接入流程

```
注册账号 → 创建团队 → 实名认证 → 创建应用 → 开发对接 → 测试调试 → 店铺授权/上架
```

## 开发者类型

| 开发者类型 | 说明 | 典型用户 |
|-----------|------|---------|
| 商家自研 | 有自研能力的商家对接内部系统 | 电商部门 IT |
| 软件服务商 | SaaS 提供商，上架应用市场 | ERP、CRM、WMS 厂商 |
| 设计/代运营 | 为商家提供技术服务 | 代理商 |
| IOT 平台 | 硬件设备与有赞数据联动 | 智能硬件厂商 |
| iPaaS | 数据集成平台 | 数据中台、BI 厂商 |

## 配套体系

- **能力包**：按业务场景打包的接口权限组（如"ERP全渠道库存同步"）
- **店铺授权**：商家在后台授权应用访问其店铺数据
- **API 套餐**：按调用量计费的套餐包，分 A/B/C 三档
- **云测试店铺**：免费测试环境，无需真实商家授权
